# 🔍 AI 스마트 모임 검색 시스템

홍밥노노 앱에 OpenAI 기반의 스마트 검색 기능이 추가되었습니다! 사용자의 자연어 검색을 AI가 분석하여 의도를 파악하고 최적의 모임을 추천합니다.

## ✨ 주요 기능

### 1. 자연어 검색 분석
- 사용자가 입력한 검색어를 AI가 분석
- 카테고리, 지역, 가격대, 시간 등의 의도 자동 추출
- 검색 필터 자동 적용

### 2. 실시간 추천
- 검색어 입력 시 AI가 실시간으로 관련 추천어 제공
- 현재 모집 중인 모임 기반 맞춤 추천
- 검색 패턴 학습을 통한 개인화된 제안

### 3. 스마트 필터링
- AI 분석 결과에 따른 자동 필터 적용
- 사용자 의도에 맞는 정확한 검색 결과
- 폴백 시스템으로 AI 미사용 시에도 기본 검색 기능 제공

## 🚀 사용법

### 1. 기본 검색
```
"강남 고기집 모임" → AI가 분석하여 강남구 + 고기/구이 카테고리로 자동 필터링
"저렴한 파티룸" → 1만원 이하 가격대 + 파티룸 카테고리로 검색
"오늘 저녁 모임" → 오늘 날짜 + 저녁 시간대로 필터링
```

### 2. 자연어 표현
```
"혼자 밥 먹기 싫어서 같이 먹을 사람 찾아요"
"회사 근처에서 점심 먹을 동료 구해요" 
"새로운 맛집 탐방하고 싶어요"
```

### 3. AI 추천 활용
- 검색어 입력 후 나타나는 AI 추천 검색어 클릭
- 검색 의도 태그 확인하여 필터 상태 파악
- 추천 검색어로 더 구체적인 검색 가능

## 🛠️ 기술 사양

### AI 서비스
- **모델**: OpenAI GPT-3.5-turbo
- **기능**: 검색 의도 분석, 추천어 생성
- **폴백**: AI 미사용 시 키워드 기반 검색

### 검색 분석 요소
- **카테고리**: 고기/구이, 전골/찌개, 피자/치킨 등 8개 분류
- **지역**: 서울시 25개 구 지원
- **가격대**: 1만원 이하 ~ 3만원 이상 4단계
- **시간**: 오늘, 내일, 주말 등 시간 표현 인식

## ⚙️ 설정 방법

### 1. OpenAI API 키 설정 (선택사항)
```bash
# .env.development 또는 .env.production 파일에 추가
REACT_APP_OPENAI_API_KEY=your_openai_api_key_here
OPENAI_API_KEY=your_openai_api_key_here
```

### 2. AI 기능 비활성화 시
- API 키가 설정되지 않으면 자동으로 기본 검색 모드로 동작
- 키워드 매칭 기반 폴백 검색 제공
- 사용자 경험에 영향 없음

## 🔧 개발자 가이드

### AI 서비스 클래스 구조
```typescript
class AISearchService {
  // 검색 의도 분석
  analyzeSearchIntent(query: string): Promise<SearchIntent>
  
  // 추천어 생성  
  generateRecommendations(query: string, meetups: any[]): Promise<string[]>
  
  // AI 사용 가능 여부 확인
  isAIEnabled(): boolean
  
  // 폴백 분석 (AI 미사용 시)

}
```
